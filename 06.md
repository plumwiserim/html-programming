## 자바스크립트(Javascript)

- 객체 기반의 스크립트 언어
  - 웹 브라우저에 인터프리터가 내장되어 주로 클라이언트 측
프로그래밍 작성
    - Node.js와 같은 프레임워크를 사용하면 서버 측 프로그래밍에서도
사용 가능
  - 웹 페이지의 동작을 구현
    - HTML 요소의 내용, 속성, 스타일 등을 변경
- 자바스크립트 코드 작성
  - 내부 자바스크립트 코드로 작성
    - `<script>` 요소로 삽입
  - 외부 자바스크립트 파일 삽입하여 작성
    - `<script src=“”>` 요소의 src 속성을 이용하여 외부 자바스크립트 파
일 추가
    - 단, 작성된 외부 자바스크립트 파일의 확장자는 .js로 사용
